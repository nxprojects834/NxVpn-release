## Cara Install

1. **Unduh** file `.apk` dari halaman [Releases](https://github.com/nxprojects834/NxVpn-release/releases/latest)
2. **Izinkan instalasi** dari sumber tidak dikenal:
   - Buka **Pengaturan** → **Keamanan** (atau **Privasi**)
   - Aktifkan **Install unknown apps** untuk browser/file manager kamu
3. **Buka** file APK yang sudah diunduh
4. Ketuk **Install** dan tunggu selesai
5. **Buka** NxVpn dan mulai terhubung
